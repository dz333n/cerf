# Known Issues Tracker

This directory tracks known visual and functional issues in CERF when running WinCE applications. Each file is named after the affected program or DLL.

## Format

Each issue file lists problems with:
- **Description**: What's wrong
- **Now**: Current behavior in CERF
- **Expected**: Correct behavior (from real WinCE device or emulator)
- **Screenshots**: Visual references when applicable (saved in `screenshots/`)

## Files

- [cecmd.exe.md](cecmd.exe.md) - Total Commander/CE
- [commctrl.dll.md](commctrl.dll.md) - Common Controls library (affects all apps)
